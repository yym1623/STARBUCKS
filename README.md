* FrontEnd - HTML5, CSS3
* server - Node
* build - Netlify

## 프로젝트 소개
> #### 시나리오
스타벅스 메인페이지를 기준으로 클론코딩

> #### 프로젝트 목표
HTML5, CSS3를 이용해서 기본적인 레이아웃 잡는 연습을 하고, JavaScript를 사용하여 DOM에 접근해본다


## 이후의 계획
HTML, CSS, JavaScript로 처음으로 정적인 사이트를 만들어봤다 이걸 계기로 이번엔 동적 사이트를 만들어보면서 JavaScript를 더 능숙하게 다룰 수 있도록 연습하자.
